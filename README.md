# Vision to Voice 📷🗣️

Vision to Voice is an Android application that empowers visually impaired users by converting visual information into spoken feedback. Using the device's camera and AI-powered image understanding, the app captures the surrounding environment, identifies objects and text, and narrates the results in real time through text-to-speech.

---

## ✨ Features

- 📸 Capture images using the device camera
- 🤖 AI-powered image analysis
- 🔊 Real-time voice output using Text-to-Speech
- 📝 Detect and read text from images
- 🎯 User-friendly interface designed for accessibility
- ⚡ Fast and responsive performance

---

## 🛠️ Tech Stack

- **Language:** Kotlin
- **IDE:** Android Studio
- **UI:** XML
- **AI:** Gemini API
- **Vision:** CameraX / Android Camera API
- **Text-to-Speech:** Android TTS Engine
- **Build Tool:** Gradle

## 🚀 Getting Started

### Prerequisites

- Android Studio (latest version)
- Android SDK
- Kotlin
- Gemini API Key

### Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/Vision-to-Voice.git
```

2. Open the project in Android Studio.

3. Add your Gemini API key.

4. Sync Gradle.

5. Run the application on an emulator or physical Android device.

---

## 📱 How It Works

1. Open the application.
2. Capture an image using the camera.
3. The image is sent to the AI model for analysis.
4. The detected scene, objects, or text are processed.
5. The result is converted into speech and played aloud for the user.
